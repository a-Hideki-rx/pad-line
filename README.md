# Pad-Line 🏥💊
**〜 電子カルテの「文字の迷宮」から、医療従事者を導く一筋の糸 〜**

Pad-Lineは、パドセブ（エンホルツマブ ベドチン）等の抗がん剤治療における**複雑な副作用（AE）発現と、介入（休薬・減量）のタイムラインを一瞬で可視化する、現場の薬剤師が開発した臨床判断支援ツール**です。

## 💡 開発の背景（Why I built this）
臨床現場のカンファレンスにおいて、以下のようなペイン（課題）がありました。
* **テキストのゴミ山:** 電子カルテの記録は「文章」や「点情報」として散在しており、患者の治療経過の全体像を把握するのに膨大な時間がかかる。
* **認知負荷の増大:** 「いつ副作用が悪化し、いつ休薬したのか」を頭の中でパズルにする作業が、本質的な治療方針の議論を阻害している。

Pad-Lineは、「統計のためのビッグデータ収集」ではなく、**「まずは目の前のN=1の患者の認知負荷を下げる（今日の家計簿）」**ことを目的としています。現場がメリットを実感して自然と入力することで、結果的に質の高い構造化データ（Event → Decision → Outcome）が蓄積していく「Pull型」のデータ収集基盤を目指しています。

## ✨ 主要機能（Features）
* **Swimmer Plot & Swimlane View:** 複数患者の全体俯瞰と、単一患者の副作用カテゴリ別の詳細表示をワンボタンで切り替え。
* **同日イベントの自動重なり回避（Jittering）:** 複数の副作用や休薬が同日に発生しても、マークが潰れずに上下に自動分散して表示。
* **継続中イベントの自動描画:** 消失時期が未定（継続中）の副作用を「→」で視覚的に表現。
* **見やすさスライダー:** 薬剤師が直感的に操作できるよう、複雑なパラメータ（グラフの段数や間隔）を裏側で隠蔽し、1つのスライダーに統合。
